# 📝 To-Do List Manager (C++)

A console-based task manager built using C++.

## 🚀 Features
- Add, view, edit, delete tasks
- Mark tasks as completed
- Priority levels (High, Medium, Low)
- Due date with validation
- Overdue detection
- Search and filter tasks
- Sorting (priority & date)
- File persistence

## 🛠️ Tech Used
- C++
- STL (vector, algorithm, file handling)

## ▶️ How to Run
```bash
g++ todo.cpp -o todo
./todo
